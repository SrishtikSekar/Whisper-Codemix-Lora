## Dataset: OpenSLR Tamil Speech Corpus

Source: https://openslr.org

This project uses a publicly available Tamil ASR dataset from OpenSLR.
Due to size and licensing constraints, raw audio files are NOT included
in this repository.

### Expected structure after download:

openslr_tamil/
├── audio/
│   ├── utt_0001.wav
│   ├── utt_0002.wav
│   └── ...
├── transcripts.txt

### Transcript format:
utt_0001 இந்த படம் மிகவும் நல்லது
utt_0002 இன்று வானிலை மிகவும் சூடாக உள்ளது

### Audio format:
- WAV
- 16 kHz
- Mono channel
